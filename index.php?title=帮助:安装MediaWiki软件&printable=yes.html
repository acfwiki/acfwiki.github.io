帮助:安装MediaWiki软件

来自AcFun Wiki

本文介绍在Windows XP操作系统下安装MediaWiki维基系统软件，并安装其他辅助软件，最终运行MediaWiki网站的流程。MediaWiki目前的版本为1.10.1。在Linux操作系统下的安装帮助文档只有英语版本，在mw:Manual:Installation，另一个英语的在Windows XP下的安装指南在mw:Manual:Streamlined Windows Install Guide。

运行环境

  • MediaWiki 1.10.1 版
  • WAMP 5.0版
    • Windows + Apache + MySQL + PHP（其中后两个是必须的，而前两个有多种选择，本页面只针对WAMP这一种组合）

目的

运行MediaWiki网站

WAMP5是一个很容易安装的windows程序，执行一个安装程序就能一次性把Apache、MySQL、PHP都安装完毕。

这里以WAMP5 1.66版本的安装进行说明。执行安装程序（例如wamp5_1.6.6.exe），即可出现下面的画面：

之后按“Next”。

接下来的版权协议，当然是接受。

图三：选择WAMP5安装目录
接下来选择WAMP5的安装目录。如图所示，大约需要60MB的硬盘空间。

目录可按自己需要来调整，这里选择了d:\wamp

图四：选择WAMP5所在的程序组
接下来选择WAMP5所在程序组。接受即可，无需调整。

接下来选择是否开机自动运行WAMP5。图中文字说明：如果选择开机运行，那么所需windows服务会自动开启；否则的话相关windows服务会在WAMP5运行时开启、在WAMP5退出时关闭。

随意选择后按“Next”。

接下来是信息确认过程，直接继续，按“Install”开始真正的WAMP5安装过程。

拷贝结束前，WAMP5提示你选择存放网页的服务器根目录所在的实际位置。

一般默认的设置为“c:\wamp\www\”（本次安装选择了d:\wamp\www），你也可以选择其他位置。

安装MediaWiki软件

  • 在mw:Download/zh中下载一个稳定版本，目前最新版本是1.10.1，解压至本地硬盘，比如“C:\MediaWiki”。
  • 选择“开始”->“程序”->“Apache HTTP Server XXX”（XXX是Apache的版本）->“Configure Apache Serve”->“Edit Apache httpd.conf Configuration File”。
    • 找到相应位置并做如下修改： 
DocumentRoot "C:/MediaWiki" #网站根目录

<Directory "C:/MediaWiki">
  • 刷新“http://localhost/”即可。

配置MediaWiki

您将看到一个提示配置MediaWiki的页面，按照提示进行配置，最后将LocalSettings.php文件移动至上一级目录即配置完成。

请享受自己开设的Wiki网站吧！
