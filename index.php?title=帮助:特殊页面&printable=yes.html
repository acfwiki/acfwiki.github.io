帮助:特殊页面

来自AcFun Wiki

特殊页面是使用“Special:”名字空间的页面。这些页面任何人都不能编辑，是由系统自动生成的。所有的特殊页面可以在Special:Specialpages找到。善用这些特殊页面可以让您在编写Wiki的时候更加得心应手。

所有用户使用的特殊页面

监视列表
Special:Watchlist : 显示你所监视的页面。当您特别关注某个页面，当它更新需要被告知的时候，可以将其加入您的监视列表。
最近更改
Special:Recentchanges : 显示用户所作的最近修改。最近修改的头部可以用Template:Recentchanges进行编辑。（Help:最近更改）
上载文件
Special:Upload : 上传文件到Wiki。只有注册用户才能使用。
图像列表
Special:Imagelist : 按照大小或日期排列的上传图像的列表。
用户列表
Special:Listusers : 所有本Wiki注册用户的列表。
管理员列表 
Special:Listadmins : 所有拥有管理员权限的用户列表(wikipedia:管理员)
统计
Special:Statistics : 用户数与文章数的统计。（wikipedia:统计）
随机页面
Special:Randompage : 随机显示一个条目。
孤立页面
Special:Lonelypages : 显示所有没有和其他文章链接的页面。
孤立图像
Special:Unusedimages ：所有没有在文章中使用的图像的列表 。
待撰页面
Special:Wantedpages : 在文章中有链接，但是还没有编写的页面。
短页面
Special:Shortpages - 按照文章的字符数从小到大排列的所有条目。
长页面
Special:Longpages - 按照文章的字符数从大到小排列的所有条目。
新页面
Special:Newpages : 按照文章创建时间排列的从近到远的所有条目，包括创建的时间，创建者，当前大小。
旧页面
Special:Ancientpages - 按照文章创建时间排列的从远到近的所有条目。
问题页面
Special:Deadendpages : 没有任何外部链接的条目。
所有页面
Special:Allpages : 所有条目的列表。
被封IP地址和用户名列表
Special:Ipblocklist : 被查封的IP地址和用户名的列表。
站外书源
Special:Booksources : 用ISBN查询一些外部网站上的书目。
分类
Special:Categories ：对页面进行分类的列表。
导出页面
Special:Export : 将特定页面或一组页面的文本以及编辑历史以XML格式导出。
版本
Special:Version : 网站当前所运行的软件的版本。
Wikimedia理事会选举
Special:Boardvote : 进行Wikimedia理事会选举的一个投票系统。

非普通用户使用的特殊页面

查封用户
Special:Blockip : 只有管理员才能使用。用来查封用户和IP。
恢复被删页面
Special:Undelete : 只有管理员才能使用。可以恢复被删除的页面及其历史。
将一名用户设为管理员
Special:Makesysop : 只有Bureaucrat才能使用。
锁定数据库
Special:Lockdb : 只有开发者才能使用。将网站设置为只读模式。
解锁数据库
Special:Unlockdb : 只有开发者才能使用。将网站设置为读/写模式。
显示进程列表
Special:ShowProcesslist : 只有开发者才能使用。
系统界面
Special:Allmessages : 显示所有MediaWiki名字空间的namespace页面。只有管理员可以修改系统界面。

当前不能使用的特殊页面

SQL查询
Special:Asksql : 只有管理员才能使用。运行数据库查询语句。（Help:数据库查询）
Unicode转换
Special:UnicodeConverter : 转换Unicode到HTML。
导入页面
Special:Import : 只有管理员才能使用。从其他wiki导入一个页面。
热点条目
Special:Popularpages : 按照浏览次数排序的条目列表。
维护页
Special:Maintenance : 几个用于日常维护的便利工具
