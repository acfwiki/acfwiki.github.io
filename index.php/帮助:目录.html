搜索

  • 进入
查找已有条目。在右上角的搜索框内输入想查询的内容，按Enter或点击“进入”即可。若您搜索的词条不存在，您将看到搜索结果。
  • 搜索
在左边的侧边栏您会发现一个搜索栏。输入您感兴趣的关键字（比如：元首）再点击“搜索”按钮，很快您就会看到所有包括该关键字的Wiki词条。
点击对应标题就可以进入查看。

编者须知

许可条款-Creative Commons Deed

  • 这是一般人可以理解的法律条款（授权条款全文）的摘要。

您可自由：

  • 共享 — 复制，分发，展示及表演本作品
  • 重混 — 创作衍生作品

惟需遵照下列条件：

  • 署名 — 您必须按照作者或授权人指定的方式对作品进行署名。
  • 相同方式共享 — 若您修改、转换本作品或以本作品为基础进行创作，

您只可采用与本协议相同或兼容的授权条款分发您的作品。

上述条件基于以下的理解：

  • 免除 — 若您获版权人许可，上述任何条件均能免除。
  • 其他权利 — 在任何情况下，本条款不会影响下列的权利：
  1. 您的公平处理或合理使用权利。
  2. 作者的精神权利。
  3. 其他人可能在作品本身或作品如何使用上有其权利，例如推广权或私隐权。
  • 通知 — 若再使用或分发本作品，您都必须向他人清楚说明本作品所适用的授权条款。

<div style="border-style:solid;border-width:1px;padding:5px;">

  • border-style:边框类型，可用变量如下：
    • none : 　无边框。与任何指定的border-width值无关
    • hidden : 　隐藏边框。IE不支持
    • dotted : 　在MAC平台上IE4+与WINDOWS和UNIX平台上IE5.5+为点线。否则为实线
    • dashed : 　在MAC平台上IE4+与WINDOWS和UNIX平台上IE5.5+为虚线。否则为实线
    • solid : 　实线边框
    • double : 　双线边框。两条单线与其间隔的和等于指定的border-width值
    • groove : 　根据border-color的值画3D凹槽
    • ridge : 　根据border-color的值画菱形边框
    • inset : 　根据border-color的值画3D凹边
    • outset : 　根据border-color的值画3D凸边
  • border-width:边框宽度，单位为像素（px）
  • padding:内容与边框的间距，单位为像素（px）

文本位置控制 
<div style="float: right;font-size:12px;background:#f0f0f0;
margin: 2.8em 1.0em 3.0em 3.2em; 
padding: 1.0em 3.5em 1.0em 2.0em">
</div>
  • float: 控制位置，可用参数right/left/centre
  • font-size:控制字体大小
  • background:背景色，请参考颜色列表
  • margin: 模块与模块的空隙，这里提供的例子有四个变量，实际使用中，可以不必全部使用：
    • 四个变量时，分别对应上、右、下、左；
    • 三个变量时，分别对应上、左右、下；
    • 两个变量时，分别对应上下、左右；
    • 单位可为px、cm、em等
  • padding: 内容与边框的空隙，用法同上。

无参数模板

当您需要使用某模板时，只需要输入{{模板名}}就可以了。

例如下面这个例子

输入 

文字1{{·}}文字2

就会出现

文字1 ·文字2

带参数模板

当某模板含有参数时，需要输入{{模板名|参数1|参数2}}就可以了。
