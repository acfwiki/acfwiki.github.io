用户登录后，可以在页面顶部看到一个“监视”的链接。点击这个链接您可以将当前页面及其对话页（或者它是一个对话页的话，连同所讨论的文章页面）添加到您的“监视列表”中。您可以点击导航条上或者特殊页面上的“监视列表”链接进入。您的监视列表相当于是一个定制的最近更改，可以直接监视您所关注的文章。他给出了一个所有被监视的页面的列表，并按照最后编辑该页的时间顺序逆序排列，并且可以选择一些时间选项。

默认的设置是在当前的监视列表上最多显示3天以内1000个页面，而对于一些维基百科狂12个小时就会有1000个以上的页面。

每行显示了一些关于最后一次编辑的详细的信息：日期，小修改或者主要的修改，时间，文章的链接，当前版本和上一次编辑的版本的差异，修订历史，用户名和编辑摘要。

请注意每个被编辑的页面只显示最后一次的修改。

在最近更改和增强的最近更改中被监视的文章会以粗体显示。因此即使不是用监视列表，也能关注到您所感兴趣的文章。

但是，被监视的页面被删除不会在监视列表中反映出来。解决的办法就是将删除纪录添加到监视列表中。

如何监视一个页面

点击所要监视页面的文字连接即可

监视一个页面的效果

登陆后可以在任何的维基百科页面都可以看到监视本页和停止监视的链接。

用户可以指定一个被编辑的页面是否必须被监视：

在编辑模式下，在小修改的复选框旁边还有一个“监视本页”的复选框。选择这个复选框并保存页面，就可以将这个文章监视起来。在查看一个被监视的页面时，“监视本页”链接上的文字就变成了“停止监视”，点击这个链接就会将这个页面及其对话页从您的监视列表中删除。

另外，您也可以从参数设置中选择“监视新的以及更改过的文章”。

监视不存在的页面

你还可以监视一个不存在的页面。当有人创建该页面时，你就可以从监视列表中看到。创建方法：

  • 如果一个页面含有坏链接，点击该链接，然后按上方的“监视”按钮。（有的界面皮肤设定，要求你先按“取消”后，才能进入监视链接）；
  • 你也可以直接通过URL，进入不存在的页面，设定监视。

保留一些单独的监视连表

有的页面，还有一些类似分页面的链接，也可以作为一组监视列表，用于监视链出更改。

Css

作为一种替代办法或除了使用监视功能，还可以定义一个用户的风格，链接到选定的网页，把在一个的CSS线的名单如下：

一个[标题= “网页” ] （颜色：白色;背景：红色;字体大小:150 ％ ） 这一工程在Opera ，而不是在IE浏览器。

关于（扩大）最近的更改网页作品，如粗体上述功能，但它的用途更加广泛，例如:允许额外的重点之一的网页上是非常感兴趣的，或不同的风格为不同类别的有趣网页。此外，它也适用于用户贡献的网页，并定期页（也为管道的联系，而不是间接的联系，通过重定向） 。它也适用于，少用地，该科编辑中的链接页面本身。

为了突出链接到特定网页还来自其他网站，包括中介环节，而不是使用上述：

一个[ href = “完整的URL ” ] （ .. ） 请注意，完整的URL是必要的，甚至强调链接来自相同的项目，虽然HTML代码使用相对URL /维基/网页。

监视分类或图像

利用“链出更改”定制监视列表

我们可以利用“链出更改”功能来定制监视列表。

利用现有的分类和列表

假设我们对列表“”里的所有条目感兴趣，当我们查看该列表的“链出更改”时，该列表内链接的所有条目的最近的更改就被罗列出来。但是要注意一个不同于系统自有的监视列表的重要之处，你无法自动监视列表中条目的讨论页。要想监视一个“讨论页”，该讨论页的链接也必需在列表中。

定制自己的监视列表

我们可以在自己的用户页面里创建一个列表，列上我们感兴趣的内容。当我们查看该列表的“链出更改”时，该列表内链接的所有页面的最近的更改就被罗列出来。要想同时监视一个“讨论页”，你必需把它的链接也加入你的列表。

关于监视列表的隐私

綠色或紅色數字代表什麼？

顯示在監視列表和最新修改頁的數字，顯示的是該頁增加或減少的位元組數量。如果增减的字节数很多的话，还会用粗体显示。

一般來說，在英文版本的维基项目中，這個數字顯示的和增減的字母數是一樣的，在中文版本的维基项目中，这个数字一般是增減汉字的三倍。正分的綠色數字 (+xx) 代表增加的位元組數量；負分的紅色數字 (-xx) 代表減少的位元組數量。

如果要從你的監視列表和最新修改頁完全移除這些數字，在你的個人化 CSS 加入以下文字： 

.mw-plusminus-pos {display: none;}
.mw-plusminus-neg {display: none;}
.mw-plusminus-null {display: none;}


如果要用黑色顯示這些數字，在你的個人化 CSS 加入以下文字： 

.mw-plusminus-pos {color: black;}
.mw-plusminus-neg {color: black;}
.mw-plusminus-null {color: black;}

最近更改的巡查人员如果希望加粗数字并用明亮的颜色，只需要将以下代码加入他们的个人CSS即可。 

.mw-plusminus-pos {
    color:green;
    font-weight:bold;
}

.mw-plusminus-neg {
    color:red;
    font-weight:bold;
}

修改结束后，需要清除缓存才会生效。
