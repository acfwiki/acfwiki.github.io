工作流程

本组自主工作，任务不组织分配，自主领取，7*24随时可以工作。时间不设定期限，闲的时候去做就可以，千万不要耽误您自己的日常生活。

我们总是感谢每一位成员做出的任何贡献，无论多少。

翻译工作流程

本字幕组采用Ac Wiki协作。主要工作场所为页面日本科学技术。

为防止撞车，请严格遵守以下的协作流程：

  1. 日本科学技术页面中，《生肉列表》的最后一列为本组所用，是翻译责任者的签名——有签名说明有人正在工作，别人不要接
  2. 请挑选一个最后一列无人签名的视频工作
  3. 请务必在开工之前编辑页面，在生肉列表的最后一列签名
  4. 一定不要忘记先签名再开工！！！这是本组避免撞车的唯一机制！！！
  5. 稿件结束后请将文本上传到群文件（也建议您自留一份底稿别删）
  • 允许一口气占几个坑慢慢做
  • 想弃坑，没问题，把自己的签名抹掉就可以
  • 只签名字即可，不用随时编辑页面更新进展，甚至于交稿后也不必
  • 不过交稿之后也要保留自己的名字啊，别抹掉，抹了就悲剧了 TAT
  • 视频链接从生肉列表里得到即可，如果是“群网盘OK”就到群网盘去下载观看
  • 有些生肉是空白，底色浅红色的行，大工会慢慢把生肉传到群网盘上去，请耐心等待

字幕格式如下：

  1. 纯文本文件，编码不限
  2. 每行一句，仅按照时间排列
  3. 空格分隔，不加标点，不加行号
  4. 允许使用顶端字幕解说
  5. 允许使用定位字幕、移动字幕等特效
  6. 任何解说和特效，请和一般字幕一起，严格按照时间混排在一起

大工工作流程

大工拿到文本之后做剩下的润色/时间轴/特效/压制/上传/投稿工作。群网盘里上传有成品节目的ASS文件，如果有别人接时间轴和特效，可以作为样板参考。如果有别人要接压制/上传/投稿，请先跟大工打个招呼。（以下是目前大工自己的制作习惯）

做稿、压制

准备
  1. 文本从“文本-未完成”复制到工作目录
  2. 视频从“电影”复制到工作目录
  3. 如果视频有黑边，立刻压制一个裁边的生肉（AegiSub用）（和文本润色一起做，节省等待时间）
润色、打轴、特效
  1. Notepad++编辑TXT文本润色（PopSub认的字符集：UCS-2 Little Endian）
  2. 润色时，插入视频头尾的提示语
  3. PopSub输入TXT文本，打轴，存ASS
  4. Notepad++编辑ASS，批量替换正文台词（淡入淡出 {\fad(500,300)} ）
  5. 把视频头尾的提示语换成正确的
  6. AegiSub做特效

打轴提示

  • 字幕出现后，或彻底消失之后，立刻按键抓时间点
  • 不要做任何的预判
  • 打完轴后，所有的时间点前移0.2s

检查
  1. 头尾提示语的Default和Small样式必须正确。
  2. 头尾提示语的译者、年份、EP号、版本号必须正确
压制
  1. CRF=20-22，极端情况可以19
  2. 裁边的命令行参数是 --vf crop:0,60,0,60
  3. 压制后，对成品重新进行一次上述的检查

上传、清理

  1. 先删除工作目录里其他的杂乱文件，只留成品的TXT、ASS和FLV
  2. 进Chrome隐私模式，上微博马甲，发微博视频（净网活动影响）
  3. 开始收集工作目录里的成品：ASS和FLV移动到“成品”，TXT移动到“文本-历史存档”
  4. 回头检查“文本-未完成”里的：TXT文本也移动到“文本-历史存档”（如果提示覆盖则选择“两个都保留”）

AB站投稿

  1. 【等待】新浪审核之后才能开始
  2. 获得新浪视频地址
  3. Photoshop做缩略图
  4. AcFun、Bilibili投稿，获得AC和AV号
  5. 立刻编辑Ac Wiki日本科学技术页面，熟肉表格增加，生肉表格上色
A站审核后
  1. 【等待】A站猴子审核之后才能开始
  2. 投稿加入“THE MAKING 日本科学技术”ac776专辑。
B站审核后
  1. 【等待】B站猴子审核之后才能开始
  2. 把投稿加入“THE MAKING”专题。
  3. 如果是201-300，则编辑B站的av1133388：EP201-300合集投稿，加入分P

翻译注意

请千万多查一查，力求准确。

这是旧番，没有人期待我们的作品快速发布。各种错误尽量不要发生（尤其是缺失关键的纯假名动词，造成意思不准，这个看目前的文本经常出现）。

  • 所有的数字部分，除特殊情况，一律使用阿拉伯数字表示
  • 所有单位均换算为公制单位，一般用汉字表示，m、mm、cm、kg等非常常用的也可用英文表示（像um、nm这样的就别英文了）
  • 温度一律简单的表示为“XX度”，不使用全称“摄氏度”，也不使用“℃”符号
  • 角度一律简单的表示为“XX度”，不适用“°”符号
  • 【XX工程(こうてい)】 一律翻译为“XX工序”。不译作“XX工程”、“XX步骤”或其他。
  • 【選別(せんべつ)】 一律翻译为“筛选”。不译作“选别”、“选择”、“区分”、“挑选”或其他。
  • 【決(き)められた】 一律翻译为“规定的”。不译作“决定的”、“预定的”或其他。
